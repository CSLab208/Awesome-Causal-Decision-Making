## Why Causality?

Traditional machine learning mainly focuses on **association** and **prediction**.  
Causal inference goes further by asking:

- Which variables are the actual causes?
- What happens if we intervene?
- What would have happened under another decision?
- How can causal knowledge support better policies?

In this sense, causality is important not only for **interpretability**, but also for:

- **robustness**
- **generalization**
- **counterfactual reasoning**
- **decision optimization**

## Learning Roadmap

This repository will be organized around three main lines:

### 1. Causal Structure
Focus: discovering causal relationships among variables.

Core question:

> **Who affects whom?**

### 2. Causal Effect
Focus: estimating the true impact of interventions or treatments.

Core question:

> **How much does an action change the outcome?**

### 3. Causal Policy
Focus: evaluating and optimizing decision rules using causal knowledge.

Core question:

> **How should we make better decisions?**
